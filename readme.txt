Easy to set-up Login, Logout and Registration manager

== Description ==

Easy to set-up Login, Logout manager (and soon for Registration)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/login-registration-logout-redirects-manager` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the 'Settings' -> 'Login/Register/Logout redirects' screen to configure the plugin
